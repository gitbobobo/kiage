# CLAUDE.md

## 项目概述

**kiage** 是一款运行在越狱 Kindle 上的自用看板应用，通过 [KUAL](https://wiki.mobileread.com/wiki/KUAL) 启动。

### 目标

- 定时查询多个 AI 服务提供商的套餐用量
- 以图表形式在 e-ink 屏幕上展示用量数据
- 支持数据持久化（历史记录、离线查看）
- 支持本地调试，确保安装到 Kindle 后功能与界面均正常

### 核心需求

| 需求 | 说明 |
|------|------|
| 多提供商 | 同时支持多个 AI 套餐来源，可扩展 |
| 定时查询 | 后台或周期性拉取用量，避免频繁手动操作 |
| 图表展示 | 在 Kindle 屏幕上以可读方式呈现趋势与占比 |
| 数据持久化 | 本地存储查询结果，断网可查看历史 |
| 本地调试 | 开发机可模拟或直连调试，不必每次都在真机上迭代 |
| Kindle 部署 | 以 KUAL 扩展形式安装，真机验证功能与 UI |
